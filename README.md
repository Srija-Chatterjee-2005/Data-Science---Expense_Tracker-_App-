# 💰 Expense Tracker Analytics Dashboard - Data Science
---

## **🚀 A. Project Explanation**
---

### **🔹 What is this project?**

This is a data-driven Expense Tracker Application that records, processes, analyzes, and visualizes financial transactions to help users understand their spending behavior.

### **🔹 Why is it important?**

Managing finances manually is inefficient. This project:

Converts raw data into meaningful insights 📊

Helps detect overspending ⚠️

Supports better financial decision-making 💡

### **🔹 Real-world usage**

**Used in:**

Personal finance apps 💳

Business expense systems 🏢

Budgeting & cost control tools 📈

### **🔹 Workflow**
Data Input → Storage → Cleaning → Analysis → Visualization → Insights → Decision Making

## **🛠️ N. Troubleshooting**
---

### **🔹 Streamlit not recognized**

**python -m streamlit run app.py**

### **🔹 CSV file not found**

**python main.py**

(This generates the dataset automatically)

### **🔹 Dashboard not loading properly**

Ensure all dependencies are installed

**Run:**

**pip install -r requirements.txt**

### **🔹 Module not found error**

Check if you're inside the project folder

Ensure src/ folder exists

### **🔹 Charts not showing**

Make sure dataset is not empty

**Re-run:**

**python main.py**

### **🔹 Git push errors**

**git push -u origin main --force**
